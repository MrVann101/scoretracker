# 🏀 Basketball Score Tracker

A real-time basketball scoring application built with **Next.js** and **Firebase**. This app allows users to track game progress, scores, and statistics in real-time.

## 🚀 Features
* **Real-time Updates:** Powered by Firebase Firestore for instant score syncing.
* **Modern UI:** Built with Next.js and styled for a clean, minimalist experience.
* **Mobile Friendly:** Designed to be used on the court via smartphone or tablet.

## 🛠️ Tech Stack
* **Framework:** [Next.js](https://nextjs.org/)
* **Database/Backend:** [Firebase](https://firebase.google.com/)
* **Deployment:** [Vercel](https://vercel.com/)
